# Portfolio - RDK.ai

A futuristic, cyberpunk-themed portfolio website for Ramachandra Dayal K, an AI Engineer specializing in Computer Vision and NLP.

## Features
- **Cyberpunk UI**: Custom scrollbars, glassmorphism, and neon aesthetics.
- **Terminal Module**: Interactive command-line interface simulation.
- **Project Showcase**: Display of recent work in AI/ML.
- **Mobile Responsive**: Adaptive layout for mobile and desktop.

## Project Structure
The project is organized as follows:
```
portfolio/
├── index.html          # Main entry point
├── assets/
│   ├── css/
│   │   └── styles.css  # Custom styles
│   ├── js/
│   │   └── script.js   # Logic and interactions
│   └── images/         # Project assets
```

## Setup & Usage
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. No build steps required (vanilla HTML/CSS/JS).

## Technologies
- **HTML5/CSS3/JavaScript**
- **Tailwind CSS** (via CDN)
- **Lucide Icons**
- **Gradio Web Component**
